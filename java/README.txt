HID API for Java

About
------

Produces Java classes and interfaces required to use HIDAPI. This project
uses BridJ to create the bindings to the C library. Thus, it requires the
BridJ jar files at runtime. Instructions can be found here: 

http://code.google.com/p/bridj/wiki/Download

Build Instructions
------

Simply run make in this directory. Requires wget, java, javac and jar in
the PATH.
